#  example create backend api with nodejs express design by oop project

## setup library
```
npm install sequelize
npm install express
npm install cors
npm install bcrypt
npm install tedious
npm install dotenv
npm install supertest 
npm install jest
npm install express-oas-generator
npm install joi
npm install -g nodemon 
npm install jsonwebtoken
```
