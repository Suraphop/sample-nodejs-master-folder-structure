#  Example create backend api with nodejs express project

## setup library
```
npm init -y
npm install sequelize
npm install express
npm install cors
npm install bcrypt
npm install tedious
npm install dotenv
npm install supertest 
npm install jest
npm install express-oas-generator
npm install joi
npm install -g nodemon 
npm install jsonwebtoken
```

## Running a program
dev mode
```
npm run dev
```
test mode
```
npm run test
```
production mode
```
npm run start
```
